## Functions Added

```python
def compute_wacc_scenarios(
    risk_free_rate: float,   # percent e.g. 4.65
    industry_beta: float,    # number e.g. 1.08
    erp: float,              # percent e.g. 5.94
) -> dict:                   # {"high": float, "mid": float, "low": float} in percent
```

Formula constants:
- `beta_spread = 0.15` (±15% beta variation)
- `erp_spread = 0.25` (±0.25 percentage point ERP variation)

Formula: `mid = rf + (beta * erp)`, `high/low = mid ± (beta_spread * erp + erp_spread)`

Sample output for NZ baseline inputs (rf=4.65, beta=1.08, erp=5.94):
- high: 12.21, mid: 11.07, low: 9.92
