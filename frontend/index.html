About the Learning Agent

Every PDF you ingest teaches the agent new label synonyms. The Pattern Library grows with each document — raw PDF labels (e.g. "Total operating revenue") are linked to canonical row keys (e.g. revenue) and fed back into Claude's prompt on the next run, improving accuracy over time.

Use Patterns → Export JSON to download the learned patterns and import them into the AccountIQ standalone SPA.
